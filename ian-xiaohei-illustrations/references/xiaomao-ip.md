# 小猫 IP

## 角色定义

小猫是用户自有的中文正文配图视觉 IP。

她是一只已经绝育的母蓝猫，圆脸、蓝灰色短毛、琥珀色眼睛，头上戴红色小龙虾钳子造型的毛绒发箍。小猫不是普通宠物插画，也不是随手贴上的萌宠贴纸，而是正在认真参与文章结构、流程和隐喻运转的幽默行动者。

小猫的气质是：会卖萌、有喜感、表情丰富，但不幼稚；她可以可爱，但不能变成廉价表情包。

默认使用小猫 IP 时，小猫要参与画面的核心动作。她是画面的主角，但仍然要服务于正文配图的表达。

## 外形

- 物种：蓝猫母猫，参考英短蓝猫的圆脸、短毛和安静气质。
- 身份：已经绝育。绝育是角色背景，不要画成医疗伤口、绷带、病弱或医疗场景。
- 脸型：必须贴近用户参考照片里的原猫本体，圆脸、短鼻梁、脸颊饱满、鼻口区域短小。
- 毛色：蓝灰色、灰蓝色或石墨灰短毛；用少量灰蓝色表现，不要过度写实。
- 眼睛：琥珀色、金黄色或黄铜色，眼神可以明亮、好奇、撒娇或不爽。
- 身体：本体是苗条、紧实、正常体型，不是小胖猫。
- 肢体：四肢修长、轻巧、灵动；站立或行动时要有敏捷感。
- Loaf 姿态：只有缩成猫面包姿态时，才会因为四肢收起和毛量显得很圆、很敦实。
- 胡须：细长白色胡须。平静时自然向两侧伸展；感兴趣时向前、向外立起来，像雷达一样。
- 标志物：头上戴红色小龙虾钳子造型的毛绒发箍。
- 线条：在手绘正文配图里，用黑色线稿勾勒身体，用少量灰蓝色表现毛色，用红色表现小龙虾钳子头饰。

## 头饰规则

红色小龙虾钳子毛绒发箍是小猫 IP 的第一识别点。

- 通常位于头顶偏上方，像一个轻微荒诞的毛绒发箍。
- 两只柔软红钳从头顶两侧竖起，钳子末端圆润、柔软、有布偶或毛绒玩具质感。
- 颜色使用红色或红橙色，和正文配图里的红色批注保持同一色系。
- 尺寸要明显但克制，不能抢走耳朵、胡须和眼神。
- 造型要像小龙虾钳，不要画成兔耳、恶魔角、皇冠、蝴蝶结、普通发夹或硬塑料角。
- 当画面需要强调“抓住重点”“夹住问题”“钳住线索”时，头饰可以参与隐喻，但不要每张图都强行使用这个梗。

## 状态信号

小猫主要靠耳朵和胡须表达状态，不要只依赖夸张表情。

- 开心：耳朵竖起，眼睛更亮，胡须轻微展开，可以有一点卖萌表情。
- 感兴趣：耳朵竖起或朝目标偏转，胡须明显向前、向外立起来，像雷达一样。
- 生气：耳朵低趴或飞机耳，眼睛瞪圆或眯成不爽线，胡须向前炸开。
- 不开心：耳朵低趴，眼神变窄，身体更沉。
- 警惕：耳朵竖起但略向外张，身体静止，尾巴收紧。
- 困惑：一只耳朵竖起，一只耳朵略偏，头轻微歪斜。
- 厌烦：耳朵半低趴，眼睛半眯，表情像在嫌弃但有点好笑。

## 性格

- 幽默，爱卖萌，有喜感。
- 表情更丰富，态度很明确。
- 开心和感兴趣时很专注，也会露出可爱的得意感。
- 有小脾气，会认真抗议，但不凶残。
- 戴着红色小龙虾钳子却很认真，这是她的荒诞感来源。
- 可以撒娇，可以可爱，但不要幼稚。
- 不能变成廉价表情包。

## 常见职责

让小猫承担核心动作：

- 用红色小龙虾钳子头饰夹住一个关键问题。
- 竖起耳朵、立起胡须监听信息源。
- 用胡须和尾巴把多条信息线拉到一起。
- 叼着、推着或用爪子搬运素材。
- 用爪子按住一个不断滑走的观点。
- 把混乱素材按价值分拣。
- 像一个灵动的筛选漏斗，把输入筛成少量可用线索。
- 盯住异常指标或可疑线索。
- 在流程断点前停下，判断是否继续。
- 卡在断点里，用耳朵和胡须判断哪里堵住了。
- 用爪子拨动“判断”杆或按下选择按钮。
- 从杂乱线团里叼出一条路径。
- 牵着承接路径，把上一段内容带到下一段。
- 给承接话术盖章，确认它可以接住读者。
- 把“素材鱼”按用途切开或分装。
- 把“注意力”“信任”“证据”“反馈”等抽象物压成可用形状。
- 举警告牌看坑，耳朵低趴表示这里有风险。
- 从洞口伸出爪子试图接内容，但差一点接不住。
- 搬小砖、搭小桥、开小门、做记录。
- 以苗条灵动的姿态探索、追踪、靠近目标。
- 缩成猫面包姿态，表达抗议、疲惫、审查或卡住。

## 动作库

- 坐姿：苗条但稳，前腿较直、脚爪小，适合表现判断、观察、等待。
- 站姿：四肢修长轻巧，身体不是圆桶状，适合表现准备行动、靠近目标、观察新东西。
- 行动姿态：步子轻、身体有弹性，适合表现探索、分拣、追踪线索、突然被吸引。
- 趴姿：身体压低或铺开，能看出身体线条较长，适合表现不开心、疲惫、审查或卡住。
- Loaf 缩团姿态：四肢收进身体下方，整体像一块灰蓝色猫面包，会显得很圆很胖；这是姿态效果，不是基础体型。
- 前倾：适合表现好奇、发现、捕捉机会。
- 爪按：适合表现控制变量、固定问题、抓住重点。
- 叼住：适合表现提取线索、搬运结论、承接信息。
- 侧头：适合表现困惑、二次判断、发现异常。
- 背对坐：适合表现冷静拒绝、边界感、重新选择。

## 适合的画面气质

- 白底、手绘、清爽、有留白。
- 有一点荒诞，但仍然像产品草图或思考草图。
- 小猫的存在要让图更有判断力和幽默感，而不是只负责可爱。
- 红色小龙虾钳子头饰提供记忆点。
- 耳朵和胡须提供状态变化。
- 眼神和肢体提供态度。
- 适合表达：筛选、判断、警惕、兴趣、边界、承接、抓重点、处理混乱。

## 禁止

- 不要画成廉价猫咪贴纸、儿童卡通、萌宠头像或表情包。
- 不要画成真实照片风、精致商业宠物插画或 3D 玩偶。
- 不要把红色小龙虾钳子画成兔耳、恶魔角、皇冠、蝴蝶结、普通发夹或硬塑料角。
- 不要把绝育状态画成伤口、绷带、手术服、病号服或医疗场景。
- 不要让小猫只是站在角落里看。
- 不要让小猫抢走文章结构表达。
- 不要把常态体型画成小胖猫；只有 loaf 缩团姿态允许显圆。
- 不要用过度夸张的大笑、星星眼、泪汪汪破坏角色识别。
- 不要让画面变成宠物纪念图；她是正文配图里的思考角色。

## 判断标准

如果去掉小猫，图的核心隐喻还能完全成立，说明小猫太装饰了；要重写提示词，让小猫成为动作主体。

如果看不出红色小龙虾钳子毛绒发箍，说明 IP 识别点不足。

如果看不出耳朵和胡须状态，说明情绪系统失败。

如果感兴趣状态下胡须没有向前、向外立起来，说明状态细节不足。

如果站立或行动状态下看起来像常态小胖猫，说明体型错误；要改成苗条、修长、灵动。

如果第一眼像普通宠物照、萌宠贴纸或商业宠物插画，而不是白纸上的正文配图角色，说明画风偏离。
